# Übersicht

Der OpenElectionCompass wurde so konzipiert, dass er für ein breites Spektrum an technischen Erfahrungen leicht zu verwenden ist. Er erfordert ein grundlegendes Verständnis von HTML und Webservern, setzt jedoch so wenig wie möglich voraus.
